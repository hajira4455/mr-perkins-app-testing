<p>
  Use <code>&lt;CardActions&gt;</code> component to create card with <code>collapse</code>, <code>refresh</code> and{' '}
  <code>close</code> actions.
</p>

### Default

---

<Alert className='my-4' color='warning'>
  <h4 className='alert-heading'>Warning</h4>
  <div className='alert-body'>
    As refresh action is used for async data fetching, you'll use <code>endLoading</code> function to false once you are
    done with fetching data.
  </div>
</Alert>

```jsx
import CardAction from '@components/card-actions'

const actions = () => (
  <CardAction
    title='Card Actions'
    actions={['collapse', 'reload', 'remove']}
    endReload={endLoading => {
      setTimeout(() => endLoading(), 2000)
    }}
  >
    ...
  </CardAction>
)
```

<p className='my-4'>
  You can check actions actions demo on{' '}
  <a href='https://pixinvent.com/demo/vuexy-react-admin-dashboard-template/demo-1/cards/action' target='_blank'>
    this
  </a>{' '}
  page.
</p>

### Specific Action

---

<p>
  You can also use only specific action if you don't want to use all actions. Just pass the name of action with{' '}
  <code>action </code>
  prop as a string and you will get that specific action.
</p>

```jsx
import CardAction from '@components/card-actions'

const actions = () => (
  <CardAction title='Collapse' actions='collapse'>
    ...
  </CardAction>
)
```
