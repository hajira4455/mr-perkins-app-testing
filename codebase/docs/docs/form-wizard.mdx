### Overview

---

<p>
  You can find the customized wizard component is <code>src/@core/components</code>
</p>

<p>Let's look at the required props you'll need to create a wizard:</p>

<ul className='pl-3 mb-5'>
  <li>
    <strong>ref:</strong> this is required to initialize the wizard
  </li>
  <li>
    <strong>steps:</strong> this should be an array of object you'll find the object properties below.{' '}
  </li>
</ul>

### Steps

---

<p>You'll need steps as an array of object for wizard to work properly.</p>

```jsx
const steps = [
  {
    id: 'account-details',
    title: 'Account Details',
    subtitle: 'Enter Your Account Details.',
    content: <AccountDetails stepper={stepper} type='wizard-horizontal' />
  }
]
```

<p className='my-4'>The object properties are listed below:</p>

        </p>
      </td>
    </tr>

      <td>Your tab title</td>
    </tr>

      <td>subtitle</td>
      <td>string</td>

      <td>Your tab subtitle</td>
    </tr>

      <td>icon</td>
      <td>any</td>

      <td>yes</td>

### Horizontal Wizard

---

<p>
  Creating a wizard is all you have to do is pass <code>ref</code> & <code>steps</code> props to wizard
</p>

```jsx title="WizardHorizontal.js"
import { useRef, useState } from 'react'
import Wizard from '@components/wizard'

const WizardHorizontal = () => {
  const [stepper, setStepper] = useState(null)
  const ref = useRef(null)

  const steps = [...]

  return <Wizard instance={el => setStepper(el)} ref={ref} steps={steps} />
}
```
