### Basic

---

<p>
  Creating an avatar is easy all you have to do is pass <code>data</code> array prop
</p>

```jsx

const data = [
  {
    img: require('@src/assets/images/portrait/small/avatar-s-5.jpg').default,
    // Any other avatar prop will work with group add them here
  },
...
]

<AvatarGroup data={data} />
```

<p className='my-4'>Result</p>

<img className='border rounded mb-5' src={useBaseUrl('img/avatar-group.png')} />

### Tooltip

---

<p>
  Use <code>title</code> to add Tooltips to avatar group.
</p>

```jsx

const data = [
  {
    title: 'Julee Rossignol',
    img: require('@src/assets/images/portrait/small/avatar-s-5.jpg').default
  },
...
]

<AvatarGroup data={data} />
```

<p className='my-4'>Result</p>
