### Badge Glow

---

<p>
  You can add shadow with badges. You have to use custom class <code>badge-glow</code>.
</p>

```jsx
<Badge pill color='primary' className='badge-glow'>Primary</Badge>
<Badge pill color='secondary' className='badge-glow'>Secondary</Badge>
<Badge pill color='success' className='badge-glow'>Success</Badge>
<Badge pill color='danger' className='badge-glow'>Danger</Badge>
<Badge pill color='warning' className='badge-glow'>Warning</Badge>
<Badge pill color='info' className='badge-glow'>Info</Badge>
<Badge pill color='dark' className='badge-glow'>Dark</Badge>
```

### Badge Light Color

---

<p>
  We have added light background color badges in our template. You have to pass <code>light-[color]</code> value to{' '}
  <code>color</code> prop.
</p>

```jsx
  <Badge pill color='light-primary'>Primary</Badge>
  <Badge pill color='light-secondary'>Secondary</Badge>
  <Badge pill color='light-success'>Success</Badge>
  <Badge pill color='light-danger'>Danger</Badge>
  <Badge pill color='light-warning'>Warning</Badge>
  <Badge pill color='light-info'>Info</Badge>
  <Badge pill color='light-dark'>Dark</Badge>
```

<img className='rounded border mb-5' src={useBaseUrl('img/pill-badge-light.jpg')} />

### Pill Badges As Notification

---

<p>
  Wrap your badge with class <code>.position-relative</code> and class <code>.badge-up</code> with your badge to create
  a notification badge. Refer code for example.
</p>

```jsx
<div className='position-relative'>
  <Badge pill color='primary' className='badge-up'>
    4
  </Badge>
  <Bell className='text-primary' size={22} />
</div>
<div className='position-relative'>
  <Badge pill color='danger' className='badge-up'>
    5
  </Badge>
  <Bell className='text-danger' size={22} />
</div>
<div className='position-relative'>
  <Badge pill color='info' className='badge-up'>
    6
  </Badge>
  <Bell className='text-info' size={22} />
</div>
```

<img className='rounded border mb-5' src={useBaseUrl('img/pill-badge-as-notification.jpg')} />

### Block Badge

---

<p>
  Use <code>.d-block</code> class with <code>&lt;Badge&gt;</code>, to display badge as a block element.
</p>

```jsx
<Badge pill color='danger' className='d-block'>
  Danger
</Badge>
```
