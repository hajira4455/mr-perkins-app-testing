### Separators

---

<p>
  You can change breadcrumb separator by using our custom classes <code>.breadcrumb-slash</code> ,{' '}
  <code>.breadcrumb-dots</code> , <code>.breadcrumb-dashes</code>, <code>.breadcrumb-pipes</code> and{' '}
  <code>.breadcrumb-chevron</code> .
</p>

```jsx
<Breadcrumb listClassName='breadcrumb-slash'>
  ...
</Breadcrumb>

<Breadcrumb listClassName='breadcrumb-dots'>
  ...
</Breadcrumb>

<Breadcrumb listClassName='breadcrumb-dashes'>
  ...
</Breadcrumb>

<Breadcrumb listClassName='breadcrumb-pipes'>
  ...
</Breadcrumb>

<Breadcrumb listClassName='breadcrumb-chevron'>
  ...
</Breadcrumb>
```
