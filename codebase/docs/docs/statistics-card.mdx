<p>This contains 4 components with different layout/UI. Let's have a glance.</p>

### Statistics Card Horizontal

---

```jsx
import StatsHorizontal from '@components/widgets/stats/StatsHorizontal'

const Component = () => <StatsHorizontal icon={<Cpu size={21} />} color='primary' stats='86%' statTitle='CPU Usage' />
```

<p className='my-3'>Result:</p>

<img className='rounded border mb-3' src={useBaseUrl('img/stats-card-horizontal.jpg')} />

<p>
  <code>&lt;StatsHorizontal&gt;</code> component renders content in two col and single row.
</p>

<p>Below is props supported by this component:</p>

<ul className='pl-3 mb-5'>
  <li>
    <strong>icon:</strong> stats icon
  </li>
  <li>
    <strong>color:</strong> color variant
  </li>
  <li>
    <strong>stats:</strong> stats to show
  </li>
  <li>
    <strong>statTitle:</strong> stats title
  </li>
  <li>
    <strong>className:</strong> card class
  </li>
</ul>

### Statistics Card With Line Chart

---

<p>
  <code>&lt;StatsWithLineChart&gt;</code> component renders content with line chart. It uses apex charts for chart.
</p>

```jsx
import StatsWithLineChart from '@components/widgets/stats/StatsWithLineChart'

const Component = () => (
  <StatsWithLineChart
    icon={<Monitor size={21} />}
    color='primary'
    stats='78.9k'
    statTitle='Site Traffic'
    series={data.series}
    type='line'

)
```
