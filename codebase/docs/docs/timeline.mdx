<p>You can use our custom timeline components to create a timeline.</p>

### Basic Timeline

---

<p>
  Creating a timeline is easy all you have to do is pass <code>data</code>.
</p>

<p>
  You can check demo on{' '}
  <a href='https://pixinvent.com/demo/vuexy-react-admin-dashboard-template/demo-1/components/timeline' target='_blank'>
    this
  </a>{' '}
  page.
</p>

```jsx
import Timeline from '@components/timeline'

const data = [
  {
    title: '12 Invoices have been paid',
    content: 'Invoices have been paid to the company.',
    meta: '12 min ago',
    customContent: (
      <Media>
        <img className='mr-1' src={pdf} alt='pdf' height='23' />
        <Media body>invoice.pdf</Media>
      </Media>
    )
  }
  ...
]

const BasicTimeline = () => <Timeline data={data} />
```

### Icons Timeline

---

<p>Add icon property in data to create a timeline with icon.</p>

```jsx
import Timeline from '@components/timeline'

const data = [
  {
    title: '12 Invoices have been paid',
    content: 'Invoices have been paid to the company.',
    icon: <PenTool size={14} />,
    meta: '12 min ago',
    customContent: (
      <Media>
        <img className='mr-1' src={pdf} alt='pdf' height='23' />
        <Media body>invoice.pdf</Media>
      </Media>
    )
  }
  ...
]

const IconsTimeline = () => <Timeline data={data} />
```
