### Default Collapse

---

<p>It is easy to create a default collapse all you have to do is pass in data.</p>

```jsx
import AppCollapse from '@components/app-collapse'

const data = [
  {
    title : "Collapse Item 1",
    content : ( <h1>Collapse Content</h1> )
  }
  ...
]

const CollapseDefault = () => <AppCollapse data={data} />
```

### Accordion Collapse

---

<p>
  Pass prop <code>accordion</code> to make your collapse an accordion.
</p>

```jsx
import AppCollapse from '@components/app-collapse'

const data = [
  {
    title : "Accordion Item 1",
    content : ( <h1>Accordion Content</h1> )
  }
  ...
]

const Accordion = () => <AppCollapse data={data} accordion />
```
